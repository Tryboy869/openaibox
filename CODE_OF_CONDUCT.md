# Code of Conduct

## Our Pledge

GraphRuntime is a tool for understanding. We are committed to keeping this
project a respectful, constructive space for everyone interested in
LLM introspection and AI transparency.

## Standards

**Expected behavior:**
- Respectful, technical discussion
- Constructive feedback on issues and pull requests
- Crediting original work and research
- Using the tool for understanding, research, and education

**Not acceptable:**
- Harassment or discriminatory language
- Using GraphRuntime analysis results to harm, manipulate, or deceive
- Misrepresenting findings or fabricating benchmark results
- Claiming authorship of others' contributions

## Scope

This Code of Conduct applies to all project spaces: GitHub issues,
pull requests, discussions, and any communication representing the project.

## Enforcement

Report violations to `anzize.contact@proton.me`.
Violations may result in removal from the project community.
